pycosec Matrix COSEC Python Library

Python library for Matrix COSEC biometric devices — configure devices, manage users and credentials, and read attendance events via the COSEC HTTP API with Basic authentication.

GitHub Repository PyPI Package

Features

Device Settings

Network, time, access, alarms, doors, enrollment, Wiegand, and smart card format.

User Lifecycle

Add, update, delete users. Face recognition toggles, credential counts and deletion.

Attendance Events

Retrieve punch-in / punch-out style events directly from the device.

Secure Transport

Plain HTTP to the device with HTTP Basic auth, as used by the COSEC API.

Installation

PyPI (recommended)

shell 
$ pip install pycosec

From source

shell 
$ git clone https://github.com/KSreethul/pycosec.git
$ cd pycosec
$ pip install .

Quick Start

python 
from pycosec import COSECBiometric

device = COSECBiometric(
    machine_ip="192.168.1.100",
    port=80,
    username="admin",
    password="password",
)

# Get user count and recent attendance events
print("Users:", device.get_user_count())
events = device.get_attendance_events(no_of_events=50)
print(events)

For the full method list, see the README on GitHub.
