pycosec

Python library for Matrix COSEC biometric devices: configure devices, manage users and credentials, and read attendance events using the COSEC HTTP API with Basic authentication.

Python 3.10+ requests ≥ 2.0 LGPL-2.1-or-later v0.1.0

Package overview

pycosec exposes a high-level client (COSECBiometric) so you can script operations that would otherwise require manual device UI or low-level API work: IP and time settings, access control, user CRUD, templates (fingerprint, card, palm, face), attendance logs, alarms, doors, and timers.

  • Device settings: network, time, access, alarms, doors, enrollment, Wiegand, smart card format, and more.
  • User lifecycle: add, update, delete; face recognition toggles; credential counts and deletion.
  • Attendance: retrieve punch-in / punch-out style events from the device.
  • Transport: plain HTTP to the device with HTTP Basic auth (as used by the COSEC API).

Installation

Install from PyPI (recommended) or from a git clone of this repository.

PyPI

pip install pycosec

From source

git clone https://github.com/KSreethul/pycosec.git
cd pycosec
pip install .

Quick start

from pycosec import COSECBiometric

device = COSECBiometric(
    machine_ip="192.168.1.100",
    port=80,
    username="admin",
    password="password",
)

print("Users:", device.get_user_count())
events = device.get_attendance_events(no_of_events=50)
print(events)

For the full method list (configuration, users, credentials), see the README on GitHub.
